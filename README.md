# Movie_Analysis_PowerBI_Project
This is a Data Analysis Project using Power BI where it shows 10 years of IMDB movie data from 2006 to 2016 by creating dash boards for movie expertiese to study the dashboards and make them easy to make decisions on movie making. 
# Power BI Movie Rating Analysis Dashboard

## Project Overview

This project presents an interactive **Power BI dashboard** for analyzing movie ratings, genres, box office performance, and trends over time. The dashboard provides data-driven insights into movie performance using advanced visualizations and KPIs.

The objective of this project is to explore relationships between **ratings, genres, revenue, and audience trends**, helping users understand movie industry patterns through interactive business intelligence reporting.

---

## Features

### Dashboard Pages

#### 1. Movie Overview & Time Trends
- Movie release trends over time
- Rating distribution analysis
- Time-based performance insights
- Historical trend visualization

#### 2. Genre & Ratings Analysis
- Genre-wise movie performance
- Rating comparisons across genres
- Popular genre insights
- Audience preferences analysis

#### 3. Top Performers & Box Office Insights
- Highest-rated movies
- Top box office performers
- Revenue vs ratings comparison
- Performance benchmarking

#### 4. Correlation & Advanced Metrics
- Relationship between ratings and revenue
- Advanced analytical insights
- Performance correlations
- Comparative metrics

---

## Key Insights
- Identify **top-performing movies** based on ratings and revenue.
- Analyze **genre popularity and audience preferences**.
- Understand **historical movie trends** over time.
- Explore **correlations between ratings and box office success**.
- Gain actionable insights through interactive filters and visualizations.
